## ⚡ Quick Start

### 📋 Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v16.0.0 or higher) - [Download](https://nodejs.org/)
- **Python** (3.8 or higher) - [Download](https://www.python.org/)
- **Git** - [Download](https://git-scm.com/)
- **npm** or **yarn** package manager

### 🔧 Installation

<details>
<summary><b>📦 Option 1: Quick Install (Recommended)</b></summary>

```bash
# Clone and setup in one command
git clone https://github.com/Abinesh-SK12/AI-Jarvis.git && cd AI-Jarvis && npm install
```

</details>

<details>
<summary><b>🛠️ Option 2: Step-by-Step Installation</b></summary>

1. **Clone the repository**
```bash
git clone https://github.com/Abinesh-SK12/AI-Jarvis.git
```

2. **Navigate to project directory**
```bash
cd AI-Jarvis
```

3. **Install Node.js dependencies**
```bash
npm install
# or using yarn
yarn install
```

4. **Install Python dependencies**
```bash
pip install -r requirements.txt
```

5. **Set up environment variables**
```bash
cp .env.example .env
# Edit .env with your configuration
```

</details>

### 🚀 Running the Application

```bash
# Development mode with hot reload
npm run dev

# Production mode
npm run build
npm start

# Run tests
npm test

# Linting
npm run lint
```

### 🐳 Docker Support

```bash
# Build Docker image
docker build -t ai-jarvis .

# Run container
docker run -p 3000:3000 ai-jarvis
```
